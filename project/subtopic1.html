What is AI and Why is it Useful and Harmful

URL What is AI?
Site Author Michigan Tech
Description This site allows you to find out more about AI. It teaches you that AI is not only a computer system that is implemented in almost all technology, but that it is also a tool that can be used for multiple purposes. It later explains how it is able to handle multiple complex problems and help simplify them to make more sense to someone. It also explains how AI is not only just an extension you can use, but it is implemented in multiple tools that are already used such as GPS, gaming, entertainment apps, social media, and many more. It also briefly goes into how AI will later become something extremely depended on and used in day to day life for certain businesses that would require it.



Url The Positives and Negatives of AI
Site Author NYU - School of Professional Studiesr
Description This site begins by allowing you to understand that while AI can be beneficial, there can also be certain downsides to it as well. We continue by learning that the pros list includes things like enhanced problem solving, information retrieval, personalization, and creativity. But we also learn that the cons are loss of authenticity, not good dependency or reliability, and multiple more things that the website continues to list. It is important to not only know the pros and cons, but to allow yourself to see both sides of the argument as to why people use or do not use AI. It is important to know how it can negatively affect you after long time usage.
